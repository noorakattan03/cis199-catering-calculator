Catering Contract Calculator

This project is a C# Windows Forms application that calculates the final cost of a catering contract based on user selections.

The application uses arrays and loops to manage data for catering companies, business clients, and contract durations. Based on user input, it applies multiple discount rules and outputs the final contract price.

Key features include:

Selection of catering company and business client using dropdown menus
Validation of user input for contract years
Application of multiple discount rules using arrays
Loop-based search logic to determine applicable discounts
Real-time calculation of final pricing

This project demonstrates skills in:

C# Windows Forms development
Arrays and data structures
Looping and conditional logic
Input validation and error handling
Event-driven programming
